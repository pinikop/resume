# My Resume

Creating this custom LaTeX class for my [CV](./PiniKoplovitchCV.pdf) stemmed from two main motivations:

1. Resume Update:
    * Why: The need for an updated version of my CV.
    * How: This LaTeX class streamlines the process, allowing quick and efficient updates.

2. Learning and Experimentation with LaTeX:
    * Why: I have no prior experience with LaTex. \
    The Desire to learn and experiment with it.
    * How: This project served as a practical learning ground, \
    enabling me to explore LaTeX's features and enhance my typesetting skills.
